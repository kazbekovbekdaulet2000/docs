---
title: "Integrations"
description: "API reference for the Integrations resource."
---

The Integrations resource lets you connect Yume Cloud to external services by configuring webhooks and API integrations. When a subscribed event occurs, Yume Cloud sends a payload to your configured URL in real time.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/integrations` | List all configured integrations |
| `POST` | `/integrations` | Create an integration |
| `GET` | `/integrations/{id}` | Get an integration by ID |
| `PUT` | `/integrations/{id}` | Update integration settings |
| `DELETE` | `/integrations/{id}` | Remove an integration |

---

## Supported events

Subscribe to any combination of the following events when creating or updating an integration.

| Event | Description |
|-------|-------------|
| `rental.created` | A new rental was created |
| `rental.completed` | A rental was completed |
| `rental.cancelled` | A rental was cancelled |
| `payment.received` | A payment was recorded |
| `invoice.overdue` | An invoice is past its due date |
| `gps.alert` | A GPS geofence alert was triggered |

## List integrations

`GET /integrations`

Returns all integrations configured for the authenticated account.

### Example request

```bash cURL
curl --request GET \
  --url https://api.yume.cloud/integrations \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{
  "data": [
    {
      "id": "int_01HZG9T0UVWXY1ZABC3DEF",
      "type": "webhook",
      "url": "https://hooks.example.com/yume",
      "events": ["rental.created", "payment.received"],
      "created_at": "2024-05-20T12:00:00Z"
    }
  ],
  "total": 1
}
```

## Create an integration

`POST /integrations`

Creates a new webhook or API integration.

### Request body

<ParamField body="type" type="string" required>
  The integration type. One of `webhook` or `api`.
</ParamField>

<ParamField body="url" type="string" required>
  The endpoint URL that will receive event payloads.
</ParamField>

<ParamField body="events" type="string[]" required>
  An array of event names to subscribe to. See [supported events](#supported-events) for the full list.
</ParamField>

<ParamField body="secret" type="string">
  An optional secret used to sign webhook payloads. When provided, Yume Cloud includes an `X-Yume-Signature` header with each request so you can verify the payload's authenticity.
</ParamField>

### Example request

```bash cURL
curl --request POST \
  --url https://api.yume.cloud/integrations \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "type": "webhook",
    "url": "https://hooks.example.com/yume",
    "events": ["rental.created", "payment.received", "invoice.overdue"],
    "secret": "my-webhook-secret"
  }'
```

### Example response

```json
{
  "id": "int_01HZG9T0UVWXY1ZABC3DEF",
  "type": "webhook",
  "url": "https://hooks.example.com/yume",
  "events": ["rental.created", "payment.received", "invoice.overdue"],
  "created_at": "2024-05-20T12:00:00Z"
}
```

## Update an integration

`PUT /integrations/{id}`

Updates an existing integration's URL, subscribed events, or secret.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the integration to update.
</ParamField>

### Request body

<ParamField body="url" type="string">
  The updated endpoint URL.
</ParamField>

<ParamField body="events" type="string[]">
  The updated list of event names to subscribe to. This replaces the existing event list entirely.
</ParamField>

<ParamField body="secret" type="string">
  The updated webhook signing secret.
</ParamField>

### Example request

```bash cURL
curl --request PUT \
  --url https://api.yume.cloud/integrations/int_01HZG9T0UVWXY1ZABC3DEF \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "events": ["rental.created", "rental.completed", "payment.received", "gps.alert"]
  }'
```

### Example response

```json
{
  "id": "int_01HZG9T0UVWXY1ZABC3DEF",
  "type": "webhook",
  "url": "https://hooks.example.com/yume",
  "events": ["rental.created", "rental.completed", "payment.received", "gps.alert"],
  "created_at": "2024-05-20T12:00:00Z"
}
```

## Remove an integration

`DELETE /integrations/{id}`

Permanently removes an integration. No further event payloads will be sent to the configured URL.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the integration to remove.
</ParamField>

### Example request

```bash cURL
curl --request DELETE \
  --url https://api.yume.cloud/integrations/int_01HZG9T0UVWXY1ZABC3DEF \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{
  "message": "Integration removed successfully."
}
```
