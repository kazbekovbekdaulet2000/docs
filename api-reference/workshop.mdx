---
title: "Workshop"
description: "API reference for the Workshop and inventory management resource."
---

The Workshop resource manages maintenance, repairs, and inspections for your assets. Use it to schedule jobs, track progress, and record completion notes.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/workshop` | List workshop jobs |
| `POST` | `/workshop` | Create a workshop job |
| `GET` | `/workshop/{id}` | Get a workshop job by ID |
| `PUT` | `/workshop/{id}` | Update a workshop job |

---

## List workshop jobs

`GET /workshop`

Returns a paginated list of workshop jobs, optionally filtered by status or asset.

### Query parameters

<ParamField query="status" type="string">
  Filter by job status. One of `pending`, `in_progress`, or `completed`.
</ParamField>

<ParamField query="asset_id" type="string">
  Filter jobs by asset ID.
</ParamField>

<ParamField query="page" type="number" default="1">
  Page number for paginated results.
</ParamField>

<ParamField query="per_page" type="number" default="20">
  Number of results per page.
</ParamField>

### Example request

```bash cURL
curl --request GET \
  --url 'https://api.yume.cloud/workshop?status=pending&page=1&per_page=20' \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{

    {
      "id": "wkp_01HZE7R8STUVW9XYZA1BCD",

      "type": "maintenance",
      "description": "Scheduled 10,000 km oil change and filter replacement.",
      "status": "pending",
      "scheduled_date": "2024-06-15",
      "notes": null,
      "completed_date": null,
      "created_at": "2024-06-01T09:00:00Z"
    }
  ],
  "page": 1,
  "per_page": 20,
  "total": 3
}
```

## Create a workshop job

`POST /workshop`

Creates a new workshop job for an asset.

### Request body

<ParamField body="asset_id" type="string" required>
  The ID of the asset to create the job for.
</ParamField>

<ParamField body="type" type="string" required>
  The type of job. One of `maintenance`, `repair`, or `inspection`.
</ParamField>

<ParamField body="description" type="string" required>
  A description of the work to be performed.
</ParamField>

<ParamField body="scheduled_date" type="string" required>
  The planned date for the job in `YYYY-MM-DD` format.
</ParamField>

### Example request

```bash cURL
curl --request POST \
  --url https://api.yume.cloud/workshop \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "asset_id": "ast_01HZB4O5PQRST6UVWXY7ZA",
    "type": "maintenance",
    "description": "Scheduled 10,000 km oil change and filter replacement.",
    "scheduled_date": "2024-06-15"
  }'
```

### Example response

```json
{
  "id": "wkp_01HZE7R8STUVW9XYZA1BCD",
  "asset_id": "ast_01HZB4O5PQRST6UVWXY7ZA",
  "type": "maintenance",
  "description": "Scheduled 10,000 km oil change and filter replacement.",
  "status": "pending",
  "scheduled_date": "2024-06-15",
  "notes": null,
  "completed_date": null,
  "created_at": "2024-06-01T09:00:00Z"
}
```

## Update a workshop job

`PUT /workshop/{id}`

Updates the status, notes, or completion date of an existing workshop job.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the workshop job to update.
</ParamField>

### Request body

<ParamField body="status" type="string">
  The updated job status. One of `pending`, `in_progress`, or `completed`.
</ParamField>

<ParamField body="notes" type="string">
  Notes about the job progress or outcome.
</ParamField>

<ParamField body="completed_date" type="string">
  The date the job was completed, in `YYYY-MM-DD` format. Required when setting `status` to `completed`.
</ParamField>

### Example request

```bash cURL
curl --request PUT \
  --url https://api.yume.cloud/workshop/wkp_01HZE7R8STUVW9XYZA1BCD \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "status": "completed",
    "notes": "Oil change and filter replacement completed. No issues found.",
    "completed_date": "2024-06-15"
  }'
```

### Example response

```json
{
  "id": "wkp_01HZE7R8STUVW9XYZA1BCD",
  "asset_id": "ast_01HZB4O5PQRST6UVWXY7ZA",
  "type": "maintenance",
  "description": "Scheduled 10,000 km oil change and filter replacement.",
  "status": "completed",
  "scheduled_date": "2024-06-15",
  "notes": "Oil change and filter replacement completed. No issues found.",
  "completed_date": "2024-06-15",
  "created_at": "2024-06-01T09:00:00Z"
}
```
