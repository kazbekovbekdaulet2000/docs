---
title: "GPS"
description: "API reference for the GPS and location tracking resource."
---

The GPS resource provides access to real-time and historical location data for tracked assets. Use it to display asset positions on a map, review movement history, or monitor active rentals.

<Info>
  GPS data availability depends on the asset having an active tracker device assigned. Assets without a tracker will not appear in location responses.
</Info>

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/gps/locations` | Get location data for an asset |
| `GET` | `/gps/locations/latest` | Get the latest location for all tracked assets |

---

## Get location data

`GET /gps/locations`

Returns location data for a specific asset. When a date range is provided, this endpoint returns the full history of locations recorded within that range. Without a date range, it returns the single most recent location only.

### Query parameters

<ParamField query="asset_id" type="string" required>
  The ID of the asset to retrieve location data for.
</ParamField>

<ParamField query="from" type="string">
  Start of the date range in ISO 8601 format (e.g. `2024-06-01T00:00:00Z`). Must be used together with `to`.
</ParamField>

<ParamField query="to" type="string">
  End of the date range in ISO 8601 format (e.g. `2024-06-07T23:59:59Z`). Must be used together with `from`.
</ParamField>

### Response fields

<ResponseField name="asset_id" type="string" required>
  The ID of the tracked asset.
</ResponseField>

<ResponseField name="lat" type="number" required>
  Latitude coordinate in decimal degrees.
</ResponseField>

<ResponseField name="lon" type="number" required>
  Longitude coordinate in decimal degrees.
</ResponseField>

<ResponseField name="speed" type="number" required>
  Speed at the time of the reading, in km/h.
</ResponseField>

<ResponseField name="heading" type="number" required>
  Direction of travel in degrees (0–360), measured clockwise from north.
</ResponseField>

<ResponseField name="timestamp" type="string" required>
  ISO 8601 timestamp of when the location was recorded.
</ResponseField>

### Example request — latest location only

```bash cURL
curl --request GET \
  --url 'https://api.yume.cloud/gps/locations?asset_id=ast_01HZB4O5PQRST6UVWXY7ZA' \
  --header 'Authorization: Bearer <token>'
```

### Example request — with date range

```bash cURL
curl --request GET \
  --url 'https://api.yume.cloud/gps/locations?asset_id=ast_01HZB4O5PQRST6UVWXY7ZA&from=2024-06-01T00:00:00Z&to=2024-06-07T23:59:59Z' \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json

  {
    "asset_id": "ast_01HZB4O5PQRST6UVWXY7ZA",
    "lat": 48.8566,
    "lon": 2.3522,

    "heading": 275,
    "timestamp": "2024-06-05T14:23:00Z"
  },
  {
    "asset_id": "ast_01HZB4O5PQRST6UVWXY7ZA",
    "lat": 48.8601,
    "lon": 2.3478,
    "speed": 0,
    "heading": 0,

  }

```

## Get latest locations

`GET /gps/locations/latest`

Returns the most recent location for every asset that has an active tracker. Optionally filter by active rental.

### Query parameters

<ParamField query="rental_id" type="string">
  Filter results to assets associated with a specific active rental.
</ParamField>

### Response fields

The response is an array of location objects. Each object contains the same fields as the `GET /gps/locations` response.

<ResponseField name="asset_id" type="string" required>
  The ID of the tracked asset.
</ResponseField>

<ResponseField name="lat" type="number" required>
  Latitude coordinate in decimal degrees.
</ResponseField>

<ResponseField name="lon" type="number" required>
  Longitude coordinate in decimal degrees.
</ResponseField>

<ResponseField name="speed" type="number" required>
  Speed at the time of the reading, in km/h.
</ResponseField>

<ResponseField name="heading" type="number" required>
  Direction of travel in degrees (0–360), measured clockwise from north.
</ResponseField>

<ResponseField name="timestamp" type="string" required>
  ISO 8601 timestamp of when the location was recorded.
</ResponseField>

### Example request — all tracked assets

```bash cURL
curl --request GET \
  --url https://api.yume.cloud/gps/locations/latest \
  --header 'Authorization: Bearer <token>'
```

### Example request — filtered by rental

```bash cURL
curl --request GET \
  --url 'https://api.yume.cloud/gps/locations/latest?rental_id=ren_01HZ8M1A2BCDE3FGHIJ4KL' \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
[

    "lon": 2.3522,

    "lon": -0.1278,
    "speed": 65.0,
    "heading": 90,
    "timestamp": "2024-06-07T09:14:30Z"
  }
]
```
