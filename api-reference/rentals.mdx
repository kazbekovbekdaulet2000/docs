Отдельного ресурса `/rentals` в экспорте OpenAPI может не быть: бизнес-логика аренды раскрывается через **CRM** — клиенты, инвентарь, группы, статусы, заказы. Используйте поиск по спецификации по тегам и путям `/v1/crm/`.

## С чего начать

- Клиенты: `GET/POST /v1/crm/clients/`
- Инвентарь: `/v1/crm/inventories/`
- Группы: `/v1/crm/groups/`
- Автопарк / штрафы (если включено): `/v1/cars/...`

## Заголовки

```
Authorization: Bearer <access>
```

## Пагинация

Часто: `page`, `pageSize`.

Уточняйте тела запросов и коды ответов в **полной** спецификации OpenAPI.
