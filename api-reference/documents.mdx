---
title: "Documents"
description: "API reference for the Documents resource."
---

The Documents resource lets you generate, retrieve, and delete documents such as invoices, contracts, and receipts associated with rentals and clients.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/documents` | List documents |
| `POST` | `/documents` | Generate a document |
| `GET` | `/documents/{id}` | Get document metadata and download URL |
| `DELETE` | `/documents/{id}` | Delete a document |

---

## List documents

`GET /documents`

Returns a paginated list of documents, optionally filtered by rental, client, or document type.

### Query parameters

<ParamField query="rental_id" type="string">
  Filter documents by rental ID.
</ParamField>

<ParamField query="client_id" type="string">
  Filter documents by client ID.
</ParamField>

<ParamField query="type" type="string">
  Filter by document type. One of `invoice`, `contract`, or `receipt`.
</ParamField>

<ParamField query="page" type="number" default="1">
  Page number for paginated results.
</ParamField>

<ParamField query="per_page" type="number" default="20">
  Number of results per page.
</ParamField>

### Example request

```bash cURL
curl --request GET \
  --url 'https://api.yume.cloud/documents?type=invoice&page=1&per_page=20' \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{
  "data": [
    {
      "id": "doc_01HZ9K2P3QRST4UVWX5YZ",
      "type": "invoice",
      "rental_id": "ren_01HZ8M1A2BCDE3FGHIJ4KL",
      "client_id": "cli_01HZ7L0Z9XYAB1CDEFG2HI",
      "url": "https://storage.yume.cloud/documents/doc_01HZ9K2P3QRST4UVWX5YZ.pdf?token=...",
      "created_at": "2024-06-01T10:30:00Z"
    }
  ],
  "page": 1,
  "per_page": 20,
  "total": 42
}
```

## Generate a document

`POST /documents`

Generates a new document and returns the document record including a download URL.

### Request body

<ParamField body="type" type="string" required>
  The type of document to generate. One of `invoice`, `contract`, or `receipt`.
</ParamField>

<ParamField body="rental_id" type="string">
  The ID of the associated rental. Required when generating rental-related documents.
</ParamField>

<ParamField body="client_id" type="string">
  The ID of the associated client.
</ParamField>

### Example request

```bash cURL
curl --request POST \
  --url https://api.yume.cloud/documents \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "type": "invoice",
    "rental_id": "ren_01HZ8M1A2BCDE3FGHIJ4KL",
    "client_id": "cli_01HZ7L0Z9XYAB1CDEFG2HI"
  }'
```

### Example response

```json
{
  "id": "doc_01HZ9K2P3QRST4UVWX5YZ",
  "type": "invoice",
  "rental_id": "ren_01HZ8M1A2BCDE3FGHIJ4KL",
  "client_id": "cli_01HZ7L0Z9XYAB1CDEFG2HI",
  "url": "https://storage.yume.cloud/documents/doc_01HZ9K2P3QRST4UVWX5YZ.pdf?token=...",
  "created_at": "2024-06-01T10:30:00Z"
}
```

## Get a document

`GET /documents/{id}`

Returns the metadata and a time-limited download URL for a single document.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the document.
</ParamField>

### Response fields

<ResponseField name="id" type="string" required>
  Unique identifier for the document.
</ResponseField>

<ResponseField name="type" type="string" required>
  Document type: `invoice`, `contract`, or `receipt`.
</ResponseField>

<ResponseField name="rental_id" type="string">
  The ID of the associated rental, if applicable.
</ResponseField>

<ResponseField name="url" type="string" required>
  A time-limited URL to download the document file.
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp of when the document was created.
</ResponseField>

<Note>
  Document download URLs are time-limited and expire after 1 hour. If the URL has expired, re-fetch the document using `GET /documents/{id}` to obtain a fresh URL.
</Note>

### Example request

```bash cURL
curl --request GET \
  --url https://api.yume.cloud/documents/doc_01HZ9K2P3QRST4UVWX5YZ \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{
  "id": "doc_01HZ9K2P3QRST4UVWX5YZ",
  "type": "invoice",
  "rental_id": "ren_01HZ8M1A2BCDE3FGHIJ4KL",
  "url": "https://storage.yume.cloud/documents/doc_01HZ9K2P3QRST4UVWX5YZ.pdf?token=eyJhbGci...",
  "created_at": "2024-06-01T10:30:00Z"
}
```

## Delete a document

`DELETE /documents/{id}`

Permanently deletes a document. This action cannot be undone.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the document to delete.
</ParamField>

### Example request

```bash cURL
curl --request DELETE \
  --url https://api.yume.cloud/documents/doc_01HZ9K2P3QRST4UVWX5YZ \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{
  "message": "Document deleted successfully."
}
```
