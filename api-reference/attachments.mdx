---
title: "Вложения"
description: "Файлы, изображения и multipart."
---

Раздел **`/v1/attachments/`** включает:

- `comments/` — вложения к комментариям
- `files/`, `images/` — загрузка файлов и изображений
- `multipart/init/`, `multipart/part/`, `multipart/complete/` — загрузка частями

Часть методов принимает **multipart/form-data** — смотрите `requestBody` в OpenAPI.

```bash
curl -X POST 'https://api.yume.cloud/v1/attachments/files/' \
  -H 'Authorization: Bearer <access>' \
  -F 'file=@/path/to/file.bin'
```
