title: "Store"
description: "API reference for the Store and product catalog resource."
---

The Store API manages products and inventory for sales operations. Use it to maintain your product catalogue, track stock levels, and update pricing.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/store` | List store products |
| `POST` | `/store` | Add a product |
| `GET` | `/store/{id}` | Get a product by ID |
| `PUT` | `/store/{id}` | Update a product |
| `DELETE` | `/store/{id}` | Remove a product |

---

## List products

`GET /store`

Returns a paginated list of products in the store.

### Query parameters

<ParamField query="category" type="string">
  Filter products by category name.
</ParamField>

<ParamField query="in_stock" type="boolean">
  When `true`, returns only products with `stock_quantity` greater than zero. When `false`, returns out-of-stock products only.
</ParamField>

<ParamField query="page" type="number" default="1">
  Page number for paginated results.
</ParamField>

<ParamField query="per_page" type="number" default="20">
  Number of results per page.
</ParamField>

### Example request

```bash cURL
curl --request GET \
  --url 'https://api.yume.cloud/store?category=accessories&in_stock=true&page=1&per_page=20' \
  --header 'Authorization: Bearer <token>'
```

## Add a product

`POST /store`

Adds a new product to the store.

### Request body

<ParamField body="name" type="string" required>
  The display name of the product.
</ParamField>

<ParamField body="description" type="string">
  A detailed description of the product.
</ParamField>

<ParamField body="price" type="number" required>
  The unit price of the product.
</ParamField>

<ParamField body="currency" type="string" required>
  The ISO 4217 currency code for the price (e.g. `EUR`, `USD`, `GBP`).
</ParamField>

<ParamField body="stock_quantity" type="number" required>
  The number of units currently in stock.
</ParamField>

<ParamField body="category" type="string">
  A category label for organising products in the catalogue.
</ParamField>

### Example request

```bash cURL
curl --request POST \
  --url https://api.yume.cloud/store \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Helmet â€” Full Face",
    "description": "DOT-certified full-face helmet, available in multiple sizes.",
    "price": 89.99,
    "currency": "EUR",
    "stock_quantity": 12,
    "category": "accessories"
  }'
```

### Example response

```json
{
  "id": "prd_01HZF8S9TUVWX0YZAB2CDE",
  "name": "Helmet â€” Full Face",
  "description": "DOT-certified full-face helmet, available in multiple sizes.",
  "price": 89.99,
  "currency": "EUR",
  "stock_quantity": 12,
  "category": "accessories",
  "created_at": "2024-05-10T08:00:00Z"
}
```

## Update a product

`PUT /store/{id}`

Updates the price, stock quantity, or description of an existing product.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the product to update.
</ParamField>

### Request body

<ParamField body="price" type="number">
  The updated unit price.
</ParamField>

<ParamField body="stock_quantity" type="number">
  The updated stock quantity.
</ParamField>

<ParamField body="description" type="string">
  The updated product description.
</ParamField>

### Example request

```bash cURL
curl --request PUT \
  --url https://api.yume.cloud/store/prd_01HZF8S9TUVWX0YZAB2CDE \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "price": 79.99,
    "stock_quantity": 8
  }'
```

```json

  "id": "prd_01HZF8S9TUVWX0YZAB2CDE",

  "stock_quantity": 8,
```

## Remove a product

`DELETE /store/{id}`

Permanently removes a product from the store. This action cannot be undone.

### Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the product to remove.
</ParamField>

### Example request

```bash cURL
curl --request DELETE \
  --url https://api.yume.cloud/store/prd_01HZF8S9TUVWX0YZAB2CDE \
  --header 'Authorization: Bearer <token>'
```

### Example response

```json
{
  "message": "Product removed successfully."
}
```
