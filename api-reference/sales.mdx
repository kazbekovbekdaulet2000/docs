---
title: "Продажи"
description: "Заказы и продажи в API."
---

Разделы продаж и заказов в Yume Cloud смотрите в OpenAPI по путям `/v1/crm/...` и связанным сущностям (заказы, позиции, доставка — в зависимости от конфигурации).

Пример заголовка:

```
Authorization: Bearer <access>
```

Конкретные эндпоинты и схемы — в файле спецификации после синхронизации (`scripts/sync-openapi.sh`).
