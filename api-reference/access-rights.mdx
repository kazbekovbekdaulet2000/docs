---
title: "Права доступа (API)"
description: "Управление доступом через API."
---

Эндпоинты для ролей и пользователей ищите в OpenAPI (агенты, группы, CRM). При отсутствии прав на операцию сервер вернёт **`403`**.

Актуальный профиль текущего пользователя:

```bash
curl https://api.yume.cloud/v1/auth/profile/ \
  -H 'Authorization: Bearer <access>'
```
