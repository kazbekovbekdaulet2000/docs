title: "Обзор API"
description: "Базовый URL, версия, пагинация и ошибки REST API Yume Cloud."
---

REST API Yume Cloud описан в **OpenAPI 3.0.3** (`info.version` в спецификации — **1.3.1**). Все пути начинаются с префикса **`/v1/`**.

## Базовый URL

```
https://api.yume.cloud
```

Пример полного пути: `https://api.yume.cloud/v1/crm/clients/`.

Допустим только **HTTPS**.

## Формат запросов

Для JSON-тел указывайте при необходимости:

```
Content-Type: application/json
Accept: application/json
```

Отдельные эндпоинты принимают `multipart/form-data` (например, вложения) — смотрите схему в OpenAPI.

## Аутентификация

После `POST /v1/auth/login/` передавайте заголовок **`Authorization: Bearer`** и ваш access-токен. Подробнее — [Аутентификация](/authentication) и [API: аутентификация](/api-reference/authentication).

## Версионирование

Версия API отражена в пути (`/v1/`). Версия спецификации документа — **1.3.1**.

## Лимиты запросов

При превышении лимита возможен ответ **`429 Too Many Requests`**. Следуйте заголовку `Retry-After`, если он есть.

## Пагинация

Для списков в спецификации часто встречаются параметры **`page`** и **`pageSize`** (в отличие от условного `per_page`). Точные поля ответа смотрите в схеме конкретного эндпоинта (например, в стиле DRF — `count`, `next`, `previous`, `results`).

Пример:

```
GET /v1/crm/clients/?page=2&pageSize=50
```

## Ошибки

Формат тела ошибки может различаться по эндпоинтам; ориентируйтесь на код HTTP и сообщение в ответе. Типичные коды:

| Код | Значение |
|-----|----------|
| 200 | Успех |
| 201 | Created |
| 400 | Неверные параметры |
| 401 | Нет или неверная аутентификация |
| 403 | Доступ запрещён |
| 404 | Не найдено |
| 422 | Ошибка валидации |
| 429 | Слишком много запросов |
| 500 | Ошибка сервера |

## Полная спецификация

Файл **`api-reference/yume-cloud-openapi.yaml`** в репозитории содержит встроенный минимальный фрагмент для Mintlify. Чтобы подставить **полный** экспорт из продукта (десятки тысяч строк), выполните:

```bash
bash scripts/sync-openapi.sh
```

(по умолчанию копируется файл из `~/Downloads/Yume Cloud API (3).yaml`).
