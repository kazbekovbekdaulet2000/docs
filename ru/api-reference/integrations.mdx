---
title: "Интеграции (API)"
description: "Подключение внешних систем."
---

Главный список: **`GET /v1/crm/integrations/`**.

Подключение / отключение:

- `POST /v1/crm/integrations/{slug}/connect/`
- `POST /v1/crm/integrations/{slug}/disconnect/`

Вендоры вынесены в отдельные ветки: `binotel`, `ringostat`, `sipuni`, `wazzup`, `wialon`, геолокация и др.

```bash
curl https://api.yume.cloud/v1/crm/integrations/ \
  -H 'Authorization: Bearer <access>'
```
