---
title: "Уведомления (API)"
description: "REST-ресурсы уведомлений."
---

Базовый путь: **`/v1/notifications/`**. Дополнительно:

- `read/`, `count/`, `archive/`
- `mobile/`, `telegram/`

```bash
curl 'https://api.yume.cloud/v1/notifications/?page=1&pageSize=20' \
  -H 'Authorization: Bearer <access>'
```

Детали полей — в схемах OpenAPI.
