Управление наборами, ценами и тарифами на инвентарь смотрите в путях **`/v1/crm/inventories/...`**, включая `sets/`, `schedules/`, `tarifs/` — см. OpenAPI.

Пример списка инвентаря:

```bash
curl 'https://api.yume.cloud/v1/crm/inventories/?page=1&pageSize=20' \
  -H 'Authorization: Bearer <access>'
```
