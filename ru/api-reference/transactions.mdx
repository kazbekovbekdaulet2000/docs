---
title: "Транзакции"
description: "Финансовые операции в API."
---

Платежи и транзакции привязаны к биллингу и CRM. Основные ресурсы смотрите в OpenAPI:

- `/v1/billing/invoices/` — счета
- `/v1/billing/cards/` — карты
- дополнительные операции оплаты и проверок — по префиксу `/v1/billing/`

```bash
curl --request GET \
  --url 'https://api.yume.cloud/v1/billing/invoices/?page=1&pageSize=20' \
  --header 'Authorization: Bearer <access>'
```
