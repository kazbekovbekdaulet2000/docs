Ключевые маршруты (см. OpenAPI):

| Ресурс | Путь (префикс) |
|--------|----------------|
| Счета | `/v1/billing/invoices/` |
| Карты | `/v1/billing/cards/` |
| Создание карты | `/v1/billing/cards/create/` |
| Расчёты / проверки | `/v1/billing/invoices/calculate/`, `check/`, `result/` |

Пример:

```bash
curl 'https://api.yume.cloud/v1/billing/invoices/?page=1&pageSize=20' \
  -H 'Authorization: Bearer <access>'
```
