Yume Cloud — облачная платформа для управления арендой и продажами. REST API даёт программный доступ к операциям: от учёта клиентов и инвентаря до платежей, GPS и документов.

## Что такое Yume Cloud

Платформа объединяет процессы арендного и торгового бизнеса. Через один API с префиксом **`/v1/`** вы работаете с данными CRM, биллингом и вложениями:

- **Клиенты и инвентарь** — клиенты, группы, единицы инвентаря, тарифы (см. `/v1/crm/...`)
- **Продажи и заказы** — в зависимости от конфигурации тенанта
- **Транзакции и биллинг** — счета, карты, расчёты (`/v1/billing/...`)
- **GPS и телематика** — устройства и поездки через интеграции (`/v1/crm/integrations/geolocation/...`, Wialon и др.)
- **Аналитика** — отчёты по данным аккаунта
- **Документы** — шаблоны и генерация (`/v1/.../documents/...` в спецификации)
- **Вложения** — файлы, изображения, multipart-загрузка (`/v1/attachments/...`)
- **Уведомления** — каналы оповещений
- **Интеграции** — Binotel, Ringostat, Sipuni, Wazzup, Wialon и другие
- **Права доступа** — роли и разграничение в рамках тенанта
- **Мастерская и склад** — заявки на сервис, состояние техники (по маршрутам в OpenAPI)

## Кому подходит

- Компании по прокату транспорта и оборудования
- Команды, которым нужны свои интерфейсы и мобильные приложения поверх Yume Cloud
- Операции, где важны автоматизация биллинга, GPS и уведомлений

## Обзор API

| Свойство | Значение |
|----------|----------|
| Протокол | REST по HTTPS |
| Формат | JSON (для тел запросов с телом — как в схеме эндпоинта) |
| Версия спецификации | **1.3.1** (см. `info.version` в OpenAPI) |
| Префикс путей | **`/v1/`** |
| Аутентификация | JWT: пара **access** / **refresh** после `POST /v1/auth/login/`; обновление — `POST /v1/auth/refresh/` |
| Базовый URL | `https://api.yume.cloud` |

В защищённых запросах передавайте **`Authorization: Bearer &lt;access&gt;`**. Подробнее — в разделе [Аутентификация](/ru/authentication).

Полный перечень путей и схем — в файле OpenAPI; для локальной разработки можно подставить полный экспорт через `scripts/sync-openapi.sh`.
