---
title: "GPS и геолокация"
description: "Устройства и координаты через интеграции."
---

В спецификации Yume Cloud геоданные доступны через интеграции CRM, в частности:

- **`/v1/crm/integrations/geolocation/devices/`** — устройства
- **`/v1/crm/integrations/geolocation/devices/{device_id}/positions/`** — позиции
- **`/v1/crm/integrations/geolocation/trips/`** — поездки
- **`/v1/crm/integrations/wialon/...`** — учётные записи и юниты Wialon

<Info>
  Координаты есть только для объектов с привязанным трекером и настроенной интеграцией.
</Info>

<Steps>
  <Step title="Получите список устройств">
    ```bash
    curl --request GET \
      'https://api.yume.cloud/v1/crm/integrations/geolocation/devices/?page=1&pageSize=50' \
      --header 'Authorization: Bearer <access>'
    ```
  </Step>

  <Step title="Запросите позиции">
    Используйте эндпоинт позиций для выбранного `device_id` (см. OpenAPI для параметров времени и пагинации).

    ```bash
    curl --request GET \
      'https://api.yume.cloud/v1/crm/integrations/geolocation/devices/123/positions/' \
      --header 'Authorization: Bearer <access>'
    ```
  </Step>

  <Step title="Связь с арендой">
    Сопоставляйте устройство с единицей инвентаря или сделкой в вашей учётной логике, используя идентификаторы из CRM.
  </Step>
</Steps>

Подробные поля ответа — только в актуальной OpenAPI.
